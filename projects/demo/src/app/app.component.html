Bem-vindo

Framework acessível para desenvolvimento web (angular 19+).

Todos os componentes seguem os princípios da WCAG:

Perceptível: texto alternativo e estrutura semântica.

Operável: suporte completo a teclado e aria.

Compreensível: feedbacks claros e foco visível.

Robusto: compatível com leitores de tela como NVDA e VoiceOver.

Accessible Button (accessible-button)

Botão acessível com suporte à navegação por teclado, foco visível e leitura por tecnologias assistivas.

Inputs:🔧

  • label: texto visível do botão
  • ariaLabel: rótulo para leitores de tela.

Output:🎯

  • (onClick): evento emitido ao clicar no botão.

Accessible Input (lib-accessible-input)

Campo de entrada acessível, com rótulo oculto visível para leitores de tela, foco visível, contraste adequado e mensagens de erro compatíveis com ARIA.

Inputs:🔧

  • label: texto visível do botão.
  • ariaLabel: rótulo para leitores de tela.
  • id: identificador único.
  • binding bidirecional com o valor digitado.
  • placeholder: texto de dica.
Este campo é obrigatório
